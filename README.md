# World of Wonders Editor (WoW Edit)


**World of Wonders (WoW)** is a 2D indie platforming adventure made in the **Godot Game Engine**. In this adventure, you play as a bunny hopping away from certain death and staying alive, while ensuring you avoid the "black plague", all while climbing a Volcano. This is an Editor for that game where you can build yourself **Custom Levels**!

## Origins
The WoW project originally started on [Godot 4.3](https://godotengine.org/download/archive/4.3-stable/).
The actual Editor was started on Wednesday March 11th, 2026 on [Godot 4.6.1](https://godotengine.org/download/archive/4.6.1-stable/).


## Editor Features
* **Custom Format Support:** Formats include: .dat(Level Data), .tlst (Tileset), .wowlv
* **Advanced Save System:** The editor saves as .dat and .tlst in an uncompressed, easy to read manner. The exported game saves as a compressed binary to save space.
* **Platform Support:** Primarily designed for **Windows**, but **Linux** should also just work.
